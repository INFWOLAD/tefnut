# Tefnut

## 功能说明
### BT管理
1. 多服务器登录/快速切换
2. 种子管理/信息查看/分类排序
3. 剪贴板磁力一键添加
4. 内嵌webview自动捕获提交磁力下载
5. ...仍在增加

### 仍在增加

## 主要框架/组件
1. nodejs
2. react native + expo
3. zustand状态管理
4. bna-ui组件库
5. lucide-react-native图标库

## 运行/构建
1. 以工作区形式打开
2. 已配置nodejs v23以上

```bash
npm install

# expo go打开
npm run start

# 本地构建启动（请预先配置android sudio/xcode)
# android
npx expo run:android 
# ios
npx expo run:ios

```
